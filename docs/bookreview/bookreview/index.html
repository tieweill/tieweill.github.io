Book Review: 关于《圣经》

Posted on 2018, Dec 26 One min read

《新约》和《旧约》都版本译本繁多。除了《旧约》中的《传道书》，其余都读的中文译本，且未记出版社。 零零散散，乱翻乱看。

【马太福音10】

我不是叫地上太平的，我是叫地上动刀兵。因为我是叫： 
儿与父生疏，      
女与母生疏，  
媳妇与婆婆生疏，    
人在世界上的敌人都是自己家里的人。
爱父母胜过爱我的，不配做我的门徒。爱子女胜过爱我，不配做我的门徒。 
为我献出生命者，能得到生命。

耶稣的这段话令人有太多的疑惑，难道博爱的耶稣此时反倒叫人来与自己的父母生疏了？ 木心先生解释：“所有有趣的小孩子在学校里走，突告母亲、姐姐送伞来，必羞臊，这是心理。 小学，性质上就是伊甸园。儿童有儿童的浪漫主义，一时出现父母，即拉回现世。天堂人间不能共存，世俗与理想难以沟通。 ”从某种意义上来说，血缘是作为一个人最大的羁绊，它可以成就人性的伟大，同时也可以成为一个人最大的束缚。 信仰和人伦时时发生冲突，这段话可能是说信仰高于人伦，高于血缘，为了真理和信仰，必须时时准备牺牲血缘。 马太福音中有这样的场景：耶稣正对众人传道。其母与兄出现，要和他说话。 他回答：“谁是我母？谁是我兄？你们是我母，是我兄。”他在布道之时升华到一个至高的境界里，母兄的出现将他拉入现世，他说上述别扭的的话辩解和自嘲。 血缘的羁绊太紧，有时甚至蒙蔽了双眼，使其看不到真理。对血缘的珍惜和真理与信仰的追寻这时构成一种对立。 这也就是为什么“一方面伟人都是为人类的，但另一方面，又是和人类决裂的。”。

【马太福音8:21】

又有一个门徒对耶稣说，主阿，容我先回去埋葬我的父亲。 8:22 耶稣说，任凭死人埋葬他们的死人，你跟从我吧。
路加 9:51（ 耶稣被接上升的日子将到，他就定意向耶路撒冷去……）
9:59 又对一个人说，跟从我来，那人说，主，容我先回去埋葬我的父亲。 
9:60 耶稣说，任凭死人埋葬他们的死人。你只管去传扬神国的道。
9:61 又有一人说，主，我要跟从你。但容我先去辞别我家里的人。
9:62 耶稣说，手扶着犁向后看的，不配进神的国。

【马太福音5】

5:38“你们听过这吩咐：以眼还眼，以牙还牙
5:39但是我告诉你们：不要与恶人作对。有人打你的右脸，把左脸也转给他；
5:40有人想要控告你，并要拿走你的里衣，你就连外衣也由他拿去；
5:41有人强迫你走一里路，你就与他一起走两里；
5:42有人求你，你要给他；有人要向你借什么，你不要拒绝
5:43—5:45要爱你的邻人，恨你的敌人。但是我告诉你们：要爱你们的敌人，为那些逼迫你们的人祷告，好让你们成为你们天父的儿女。因为他使太阳升起，对着恶人，也对着好人；降雨给义人，也给不义的人。

这些话在一个从小耳濡目染“善有善报，恶有恶报”观念的人看来是极佩服而又极不舒服的。 耶稣使信徒相信有天堂有地狱，有善也有恶，这已然作出判断，贴上了标签，而今又说太阳照着好人也照着坏人，雨既降给义人，也降给不义之人。 这就成为一种矛盾，如果这样，最后的审判哪里来，又有何意义。 倒不如李耳说的痛快：天地不仁，以万物为刍狗！何来真假，道德，是非？ 上帝无是无非，无黑无白，超越善恶，令信徒听来平静而又无望。 佛教同样，讲佛博爱，空，而后空空，所谓“色即是空，空即是色”，却又种下善因善果、恶因恶果。 因而老子的是哲学，耶稣和佛创的是宗教。

【旧约·传道书】

写的很美，于我而言可以用一句话概括：都是虚空，都是捕风，日光之下并无新事。

可能读的最多最喜欢的是这一段，因为短、语言更混沌，断章取义起来毫无心理压力。觉得生活黑暗无望了就默念：“凡事都有定期，天下万务都有定时。 生有时，死有时。栽种有时，拔出所栽种的，也有时。 杀戮有时，医治有时。拆毁有时，建造有时。 哭有时，笑有时。哀恸有时，跳舞有时。……撕裂有时，缝补有时。静默有时，言语有时。“

念着念着，就能平静下来。

【旧约·约伯记】

上帝与撒旦之间的类似打赌式的斗气是导致约伯苦难的直接原因。 他们关于约伯敬畏上帝是否存在原因，或者说是否存在利害关系这一问题产生分歧，因而通过考验约伯来证明孰对孰错。 耶和华问撒旦说：“你曾用心查看我的仆人约伯没有？地上再没有人像他完全、正直、敬畏上帝、远离恶事。” 撒旦回答耶和华说：“约伯敬畏上帝，岂是无故呢？你岂不是四面围上篱笆维护他和他的家，并他一切所有吗？ 他手做的都蒙你赐福：他的家产也在地上增多。你且伸手毁他一切所有的；难道他不当面‘祝福’你！（‘祝福’在此做咒诅解）耶和华对撒旦说：“凡是他所有的都在你手中；只是不可伸手加害于他。” 耶和华这样撒手将约伯交给撒旦，存在两种可能的答案： 其一，耶和华洞察一切，他并不怀疑约伯的虔诚，只是为了是撒旦也同样信服这一点。 其二，耶和华听了撒旦的话之后，他自己也同样对约伯产生怀疑。然而参看《创世纪》22:1-19中也曾“试探”过亚伯拉罕对他的敬畏是否真诚无私。 可见，在旧约中，上帝对人心并非洞若观火，因而第二种答案可能性更大。 所以说，有没有可能撒旦与上帝是一体的——撒旦与耶和华的冲突正是耶和华自身的内在冲突。
